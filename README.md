# Chronus Sports front-end repository

Project in progress.

## Tech stack

- Next.js 14
- Tailwind
- Framer motion
- Storybook
- Jest
- React-hook-forms
- Zod
- Zustand
- embla-carousel
- class-variance-authority

## How to Use

Clone and run the project (requires Node 18 or higher)

```bash
npm i && npm run dev
```

## Running Tests

```bash
npm test
```

## Running Storybook

```bash
npm run storybook
```
